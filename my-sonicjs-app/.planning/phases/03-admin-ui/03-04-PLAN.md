   ```html
   <input
     type="text"
     name="search"
     placeholder="Search source or destination..."
     value="${data.filters.search || ''}"
     class="form-input"
     oninput="debounceSearch(this.value)"
   >
   ```

5. Add debounce search script:
   ```javascript
   let searchTimeout;
   function debounceSearch(value) {
     clearTimeout(searchTimeout);
     searchTimeout = setTimeout(() => {
       const params = new URLSearchParams(window.location.search);
       if (value) {
         params.set('search', value);
       } else {
         params.delete('search');
       }
       params.set('page', '1'); // Reset to page 1
       window.location.href = window.location.pathname + '?' + params.toString();
     }, 300);
   }
   ```

6. Add filter chips section (appears between filter bar and table):
   - Create renderActiveFilterChips(filters) helper function
   - For each active filter (status, matchType, isActive, search), render a chip:
     ```html
     <span class="inline-flex items-center gap-x-1 rounded-md bg-blue-50 dark:bg-blue-900/20 px-2 py-1 text-xs font-medium text-blue-700 dark:text-blue-300">
       ${filterLabel}: ${filterValue}
       <button onclick="removeFilter('${filterName}')" class="...">
         <svg>X icon</svg>
       </button>
     </span>
     ```
   - Include "Clear all" button if any filters active

7. Add JavaScript for filter chip removal:
   ```javascript
   function removeFilter(name) {
     const params = new URLSearchParams(window.location.search);
   ```
